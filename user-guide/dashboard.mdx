## Overview

The Defensoría Civil operator dashboard provides a comprehensive interface for managing divorce cases, reviewing documents, monitoring metrics, and communicating with users. Built with React, TypeScript, and TanStack Query for real-time updates.

<Note>
  The dashboard auto-refreshes metrics every 30 seconds to provide up-to-date case statistics.
</Note>

## User Management

**Note:** Currently operator accounts are created via backend script. Future versions will include UI.

### Authentication

**Login route:** `/login`

**Authentication flow:**
1. User enters username and password
2. `POST /api/auth/login` with credentials
3. Backend validates against `users` table
4. Returns JWT token + user data
5. Token stored in `authStore` (Zustand)
6. Token sent in `Authorization: Bearer {token}` header

**Auto-refresh:** Token refresh on 401 responses

**API integration:**
```typescript
export const casesApi = {
  getAll: (filters: CaseFilters) => 
    axios.get('/api/cases', { params: filters }),
  getById: (id: number) => 
    axios.get(`/api/cases/${id}`),
  downloadPetition: (id: number) => 
    axios.get(`/api/cases/${id}/petition/pdf`, { responseType: 'blob' }),
  sendOperatorMessage: (id: number, text: string) => 
    axios.post(`/api/cases/${id}/operator-message`, { text }),
};
```
